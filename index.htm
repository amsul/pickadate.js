pickadate.js

A responsive and lightweight jQuery dateinput picker (read the docs)

Weighs 7.98kb (3.3kb gzipped) – whereas jQuery UI datepicker is 30kb minified

As easy as: $( '.datepicker' ).pickadate()

Download pickadate.js

Browser support: IE9+, Chrome, Firefox, Safari 3+, Opera 9.5+, iOS Safari, Android browser

Read the docs or view it on Github
